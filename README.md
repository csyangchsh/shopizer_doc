#Shopizer分析

[Shopizer](http://www.shopizer.com/)是一个开源的销售管理(sales management)软件，采用LGPL V2开源许可。主要功能包括：

* Product catalogue
* Shopping cart
* Order fulfillment
* Content management
* Payment management
* Shipping management

技术栈

* Spring MVC
* Spring Core
* Spring Security
* Hibernate
* JBoss Infinispan
* Elasticsearch

github地址 https://github.com/shopizer-ecommerce/shopizer

Build说明 http://www.shopizer.com/documentation.html#!/
